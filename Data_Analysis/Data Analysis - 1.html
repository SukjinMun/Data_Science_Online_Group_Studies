Data Frame¶

  • 데이터가 틀에 맞춰서 저장된 형태, 데이터를 가로 x 세로 직사각형 모양으로 틀을 갖춰 저장
  • 스프레드 시트, 테이블, 엑셀, 데이터 프레임 분야마다 다르게 불림
  • 가로(행)를 row라 하고, 세로(열)를 column이라 함
  • 한 row 안에 데이터는 Dictionary 처럼 저장
  • 한 column 안에 데이터는 List 처럼 저장
  • 한 데이터 프레임은 row들이 List 처럼 저장

Series¶

  • 데이터가 순차적으로 나열된 1차원 배열 형태
  • Index는 Value와 1:1 대응됨, Index는 Value의 위치를 나타내는 역할
  • 키(k)와 값(v)이 '{k:v}' 형태로 짝을 이루는 파이썬의 Dictionary와 비슷한 구조를 갖음
  • Dictionary의 k -> Series의 Index로,
  • Dictionary의 v -> Series의 Value로 변환해서 사용

Data Frame¶

  • 행과 열로 만들어지는 2차원 배열 구조
  • 여러 개의 Series들이 모여서 Data Frame을 이루는 구조
  • Data Frame의 열은 각각 Series 객체로, Data Frame은 여러 개의 Series들이 같은 행 Index를 기준으로 줄지어 결합된 2차원 행렬(matrix)임
